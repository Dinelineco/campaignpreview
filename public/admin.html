Endpoints

      

        
GET/api/campaignsList all campaigns

        
POST/api/campaignsCreate campaign

        
GET/api/campaigns/:idGet campaign details

        
PUT/api/campaigns/:idUpdate campaign data

        
DELETE/api/campaigns/:idDelete campaign

        
POST/api/campaigns/:id/sendSend for review

      

      

        All endpoints accept and return JSON. Campaign data includes headlines, longHeadlines, descriptions, meta, dishio, videos arrays. Each item has text, approved, and notes fields.
